API testing

API testing is a type of software testing that focuses on the API of an application. It involves verifying that the API behaves as expected and that it meets the design specifications.

Key aspects of API testing:

  • Make sure the API is working as expected
  • Security: Tests the API's security features, such as authentication and authorization.
  • Handle the request that cannot be handled by the UI, such as the request that requires the specific header or body.
  • Monitor the changes of the API response in real-time.
